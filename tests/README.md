# AI Interview System - Testing & Evaluation Artifacts

This folder contains sample test cases, input/output JSON, and evaluation metrics
for the AI-Powered Interview Emotion & Feedback Analyzer (B.Tech Final Year Project).

## Structure

- `sample_input_output/` - Sample API request/response JSON for documentation
- `test_cases.md` - Test case descriptions for manual/demo testing
- `evaluation_metrics.md` - Accuracy and comparison metrics (academic demo)
